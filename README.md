# Portfolio

В данном репозитории приведены работы для обзора  
Для каждого проекта не забудьте установить зависимости:
```python -m pip install -r requirements.txt```  

### Проекты
* **SmartAssistent** (умный помошник)  
Здесь приведены только скриншоты, т.к. исходники являются интеллектуальной собственностью заказчика.  
_Представлено в ознакомительных целях, использование без указания источника запрещено_.  
Данный проект является умной системой отображения данных и оповещения оператора
    - На основе моделей машинного обучения анализирует важность событий  
    - Предоставляет пользователю большой набор дашбордов и анализаторов исторический данных и данных реального времени
    - Упрощает навигацию и мониторинг установки

* **Reliability Assessment** (оценка надежности)  
Проект представляет собой небольшой пример прикладной программы
    - В данном проекте реализован простой интерфейс на основе PyQt и matplotlib
    - Программа один:
        - Расчет функции надежности для разных распределений в зависимости от коэффициент вариации
    - Программа два:
        - Расчет стационарного коэффициента надежности для системы с холодным резервом, для различных  
        систем с разными функциями надежности (при различных коэффициентах вариации)

* **Symbloic Regression** (символьная регрессия)  
Начальная реализация символьной регрессии для решения задач восстановления регресси.  
Плюсом данного метода по сравнению с остальными, что он позволяет находить аналитический вид зависимости
